– 按照一列或者多列升序排序
– 可以按照“不查询”的列排序
– 先按照column2排，再按照column5排
SELECT column1, column2, column3
FROM dataset
ORDER BY column2, column5</p>

<p>– 降序
– DESC只作用于最靠近的唯一一列，即column2，不作用于column5
SELECT column1, column2, column3
FROM dataset
ORDER BY column2 DESC, column5</p>

<p>–使用查询列相对编号
SELECT column1, column2, column3
FROM dataset
ORDER BY 2, column5
